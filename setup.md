# Initial Server Setup

## Creating the Virtual Machine

A cloud virtual machine was created using DigitalOcean.

- Ubuntu Server was selected
- Public IP was enabled
- SSH access was enabled

## Connecting to the Server

The server was accessed remotely using SSH.

## Apache Web Server Installation

Apache2 was installed on the Ubuntu server using the following command:

sudo apt install apache2 -y

After installation, the Apache service was tested using the server public IP address.

## Website Deployment

A basic HTML portfolio website was created and deployed on the Apache web server.

The website includes:

- About Me section
- Skills section
- Contact information

The website files were stored in the Apache web directory.
